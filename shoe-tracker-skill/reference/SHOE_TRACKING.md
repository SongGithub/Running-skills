## Mileage Lifecycle

How to interpret shoe mileage for recommendations:

| Phase | KM range | What it means | Recommended use |
|-------|----------|---------------|-----------------|
| 🆕 **Fresh** | 0-150km | Foam fresh, grip new | Any run type |
| ✅ **Sweet spot** | 150-350km | Broken in, best performance | Ideal for race/speed work |
| ⚠️ **Aging** | 350-500km | Cushion degrading | Daily training only, not races |
| 🔄 **Retire soon** | 500-650km | Foam flat, injury risk up | Recovery runs only |
| ❌ **Retired** | 650km+ | Replace | Don't run in these |

## How to Suggest Shoes

When Song asks "what shoes should I wear today?":

1. Check today's planned workout type (recovery, easy, long run, tempo, intervals, race)
2. Check which shoes in the right category have **fresh mileage**
3. If multiple options, prefer shoes with lower km
4. Mention why — "EVO SL is fresh at 93km, perfect for today's easy run"
5. Don't suggest retired or nearly-retired shoes for hard efforts
