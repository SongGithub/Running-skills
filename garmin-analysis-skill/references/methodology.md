# 分析方法论

源自 [tracing-run](https://github.com/yuchong-li/tracing-run) (yuchong-li)。

## Pa:HR Decoupling
EF = 距离/平均心率，Pa:HR = (EF后半 - EF前半) / EF前半 × 100%
阈值: <5%良好 / 5-8%边界 / >8%超出

## 力学衰减
四件套(步频↓+步距↑+GCT↑+VR↑)同时变差 = 坏形变红旗

## 结构对比元规则
1. 疲劳坏形变筛查 — 同配速前后段
2. 同强度块内疲劳 — 推段内前半vs后半
3. 末段端对端粗筛 — 首km vs 末km
